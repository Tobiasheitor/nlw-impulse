# Commands used
Run node server application
```
npm run dev
```
Run the prisma studio
```
npx prisma studio
```
For test
```
npm install jest -D
npm install ts-node -D -- for jest understand files in format typescript
```
For run tests
```
npm i -D @swc/jest
npm i @types/jest -D -- for typescript declarations
```
Run a test
```
npm run test
```
